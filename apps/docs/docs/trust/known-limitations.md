# Known limitations

The OptimIEra Rules Engine is deterministic local optimization. It is not external-model inference and its scores are product heuristics, not universal benchmark guarantees.

No 0G runtime calls, contracts, Agentic ID, payment execution, DA infrastructure, marketplace, scheduled monitoring, production deployment, or public certificates are implemented.

SIWE wallet authentication remains blocked pending compatible wallet-stack verification.

Users should review optimized prompts before production use.
