# How OptimIEra works

Current local flow:

1. Collect the raw prompt and task context.
2. Analyze intent, weaknesses, ambiguity, contradictions, safety, privacy, and injection risk.
3. Score quality dimensions with documented deterministic weights.
4. Generate Balanced, Accuracy-focused, and Token-efficient candidates.
5. Evaluate original plus candidates.
6. Select a recommendation and report ties honestly.
7. Display a prompt diff and rationale.
8. Save the selected candidate as a new immutable encrypted PromptVersion.

Future decentralized flow adds encrypted evidence storage, hash/provenance registration, Agentic ID association, and certificates.
