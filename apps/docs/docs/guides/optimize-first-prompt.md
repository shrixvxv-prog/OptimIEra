# Optimize a first prompt

Status: **COMPLETE for local deterministic optimization**.

1. Sign in to OptimIEra Studio.
2. Create or open a workspace project.
3. Open `/app/optimize`.
4. Select an existing prompt or choose **Create new prompt**.
5. Enter the raw prompt and task context.
6. Run optimization.
7. Review analysis, scores, three candidates, recommendation, and prompt diff.
8. Save the selected candidate as a new immutable encrypted PromptVersion.

The active provider label is:

```text
Provider: OptimIEra Rules Engine
Mode: Deterministic local optimization
```

This is not external-model inference. Review results before production use.
